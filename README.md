# Curso TMW Git \& GitHub 2025

Curso para iniciantes aprenderem a trabalhar com versionamento de códigos e repositórios remotos com Git e GitHub

Final do curso GitFlow e VisualStudioCode

## Fluxo de Trabalho Git local

01. git clone <endereco do projeto>
02. git checkout -b <nova_branch>
03. alterações de arquivos
04. git status
05. git add <arquivos>
06. git status
07. git commit -m "nova mensagem"
08. git push origin <nova_branch>
09. abril Pull request no GitHub para main
10. excluir <nova_branch> origin
11. git checkout main
12. git branch -D <nova_branch>

## Fluxo de trabalho GitHub <> Local (projetos open-source)

01. Fork do projeto para seu perfil GitGub
02. git clone <endereco_projeto_fork>
03. git checkout -b <nova_branch>
04. alteracoes de arquivos
05. git status
06. git add <arquivos>
07. git status
08. git commit -m "nova mensagem"
09. git push origin <nova_branch>
10. abril Pull request no GitGub da branch fork para a main do projeto original
11. excluir <nova_branch>
12. git checkout main
13. git branch -D <nova_Branch>
